Time of Day

A JavaScrip Project by Sharon Nguyen

Greeting goes here

This website uses responsive JavaScript to change different elements of the page, dependent on the time of the day. The timeframes are as follows: morning (5am-12pm), afternoon (12pm-6pm), evening (6pm-11pm), and night (11pm-5am).

Dependent on the time frame, the following website components will change: background color, greeting, and image. The images I chose are different types of Hong Kong foods, eaten at different times of the day!

This is all done by using JavaScript's Date()method, by declaring it as the variable, myHour. I then targeted HTML elements through classes to reference to in my if statements.

Feel free to come back to this website at different times of the day!
